## ğŸ”§ Troubleshooting

### **Problem: UI tidak muncul**
**Solusi:**
- Pastikan script sudah di-execute dengan benar
- Cek console (F9) untuk error messages
- Pastikan game sudah fully loaded sebelum execute
- Coba execute ulang script

### **Problem: Script tidak mendeteksi ikan secret**
**Solusi:**
- Script ini adalah template, perlu disesuaikan dengan struktur game
- Gunakan Developer Console (F9) untuk inspect objects
- Cek nama objek ikan secret di game
- Update daftar `secretFishes` di script sesuai nama yang benar

### **Problem: Multiget tidak bisa attach**
**Solusi:**
- Pastikan Roblox sudah terbuka dan game sudah dimulai
- Tutup semua executor lain yang mungkin sedang berjalan
- Restart Multiget dan Roblox
- Pastikan Windows Defender/Antivirus tidak memblokir Multiget

### **Problem: Script error saat execute**
**Solusi:**
- Cek console (F9) untuk melihat error message
- Pastikan semua kode script sudah ter-copy dengan lengkap
- Pastikan tidak ada karakter yang hilang atau rusak
- Coba paste ulang script dari file

### **Problem: Ikan secret terdeteksi tapi tidak bisa di-click**
**Solusi:**
- Cek jarak player dengan ikan secret (harus dalam `maxDistance`)
- Update `maxDistance` di config jika perlu
- Cek metode interaksi yang digunakan game (ClickDetector, RemoteEvent, dll)
- Update fungsi `autoClickSecret()` sesuai metode game
