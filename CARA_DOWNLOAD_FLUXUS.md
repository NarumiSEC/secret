## ‚úÖ Cara Download Fluxus yang Benar

### **Langkah 1: Cari Link Resmi Fluxus**

**JANGAN download dari link sembarangan!** Pastikan download dari:
- Website resmi Fluxus
- Forum/community yang terpercaya
- GitHub official (jika ada)

**Cara cari:**
1. Google: "Fluxus Roblox executor download"
2. Cari link yang terpercaya (biasanya dari forum atau website executor)
3. **HATI-HATI** dengan link yang mencurigakan

### **Langkah 2: Download dengan Multiget**

1. Buka **Multiget**
2. Klik **"Job"** ‚Üí **"New Job"** (atau shortcut)
3. Di dialog:
   - **URL**: Paste link download Fluxus yang benar
   - **Save To**: Pilih folder (misal: `C:\Users\natsu\Downloads` atau `C:\Users\natsu\Documents`)
   - **Referer**: Bisa dikosongkan
4. Klik **"OK"**
5. Tunggu download selesai

### **Langkah 3: Verify File yang Didownload**

**SEBELUM install, cek dulu file yang didownload:**

1. Buka folder tempat file disimpan
2. Cek **nama file**:
   - ‚úÖ **Benar**: `Fluxus.exe`, `Fluxus.zip`, `Fluxus.rar`, atau nama yang mengandung "Fluxus"
   - ‚ùå **Salah**: `RobloxInstaller.exe`, `RobloxPlayerLauncher.exe`, atau nama yang mengandung "Roblox"

3. Cek **ukuran file**:
   - Fluxus biasanya beberapa MB (5-50 MB)
   - Jika file terlalu kecil (< 1 MB) atau terlalu besar (> 100 MB), mungkin salah

4. Cek **tipe file**:
   - Fluxus biasanya `.exe` atau `.zip`/`.rar`
   - Jika file `.msi` atau installer Roblox, itu SALAH

### **Langkah 4: Extract (Jika File ZIP/RAR)**

Jika file yang didownload adalah `.zip` atau `.rar`:

1. Klik kanan file ‚Üí **Extract** atau **Extract Here**
2. Buka folder hasil extract
3. Cari file `Fluxus.exe` atau file executable lainnya
4. **JANGAN langsung double-click!** Baca langkah berikutnya

### **Langkah 5: Scan dengan Antivirus (Opsional tapi Recommended)**

1. Klik kanan file Fluxus ‚Üí **Scan with Windows Defender** (atau antivirus kamu)
2. Jika terdeteksi sebagai virus, bisa jadi:
   - False positive (banyak executor terdeteksi sebagai virus, ini normal)
   - Atau memang file berbahaya
3. Gunakan judgment kamu sendiri

### **Langkah 6: Install/Jalankan Fluxus**

**Fluxus biasanya TIDAK perlu install**, langsung jalankan:

1. Klik kanan `Fluxus.exe` ‚Üí **Run as Administrator**
2. Atau double-click (jika tidak perlu admin)
3. Fluxus akan terbuka dengan interface executor

**Jika muncul error atau tidak bisa dibuka:**
- Cek apakah Windows Defender memblokir
- Coba disable antivirus sementara (hati-hati!)
- Atau tambahkan ke exception Windows Defender

## üö® Troubleshooting

### **Problem: File yang didownload selalu jadi Roblox**

**Solusi:**
1. **Cek link download** - mungkin link yang salah
2. **Download dari sumber lain** - coba dari website/forum berbeda
3. **Cek apakah link redirect** - beberapa link bisa redirect ke Roblox
4. **Gunakan browser langsung** - coba download pakai browser, bukan Multiget

### **Problem: Fluxus tidak bisa dibuka**

**Solusi:**
1. **Run as Administrator** - klik kanan ‚Üí Run as Administrator
2. **Disable Windows Defender sementara** - tambahkan exception
3. **Cek apakah file corrupt** - download ulang
4. **Cek system requirements** - pastikan Windows kamu compatible

### **Problem: Windows Defender memblokir Fluxus**

**Solusi:**
1. Buka **Windows Security** ‚Üí **Virus & threat protection**
2. Klik **Manage settings** di bawah "Virus & threat protection settings"
3. Scroll ke bawah ‚Üí **Exclusions** ‚Üí **Add or remove exclusions**
4. Klik **Add an exclusion** ‚Üí **File** atau **Folder**
5. Pilih file/folder Fluxus
6. Fluxus seharusnya bisa dijalankan sekarang
